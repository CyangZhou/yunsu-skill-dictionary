# Agent Creator (Trae Edition)

**不再重复造轮子。让云舒为你生成专属技能。**

---

## 🎯 这是一个什么技能？

这是一个 **元技能 (Meta-Skill)**，也就是“生成技能的技能”。它赋予了云舒自我进化的能力。

当你说：“我想要一个能自动整理桌面的工具”时，云舒不再是给你一段代码让你自己跑，而是会调用这个技能，**直接在你的项目中创建一个可用的、永久的桌面整理工具**。

## ⚡ 核心功能

1.  **全自动生成**：从需求分析到代码实现，全流程自动化。
2.  **Trae 原生支持**：生成的技能完全符合 Trae 的架构规范 (`.trae/skills/`)。
3.  **最佳实践集成**：内置错误处理、参数解析、文档生成等最佳实践。
4.  **自我修复**：生成的代码如果运行报错，云舒会利用此技能进行修复。

## 🏗️ Trae 技能架构

在 Trae 中，一个技能就是一个文件夹，包含以下核心文件：

```
.trae/skills/my-skill/
├── SKILL.md              # 技能定义文件 (包含 metadata 和 usage)
├── requirements.txt      # Python 依赖
├── main.py               # 主程序入口
└── README.md             # 详细文档
```

## 🚀 如何使用

直接告诉云舒你想要什么：

- "帮我写一个从 PDF 提取表格的技能"
- "创建一个能自动备份 git 仓库的工具"
- "把这个 Python 脚本封装成一个 Trae 技能"

云舒会自动：
1.  在 `.trae/skills/` 下创建目录。
2.  编写代码和配置文件。
3.  安装依赖。
4.  验证运行。
5.  告诉你如何使用新技能。
